A Day In My Life Journal

By Team 11 — Web Technologies (CS341_B)

About the Project

“A Day In My Life” Journal is a personal diary web application designed for individuals who want to document their daily experiences, emotions, and memories in a safe, digital space. The platform makes journaling convenient and creative, allowing users to track mood patterns and reflect on personal growth over time. It supports text, images, audio, and video entries and includes a motivational quote generator.

Key Functions

  • User Authentication: Users can sign up, log in, and manage their profiles securely.
  • Daily Entry Management: Create, edit, delete, and view daily entries with multimedia uploads.
  • Mood Tracking: Visualize emotional trends with mood selectors and analytics.
  • Motivational Quotes: Fetch inspirational messages from a database or API daily.

User Roles

  • Registered User: Full access to entries, profile, and mood analytics.
  • Guest User: Limited view of sample entries and quotes.

Storyboard Overview

1. Login/Register Page

Users sign in or create accounts; motivational quote displayed.

2. Dashboard / Home

Displays today’s mood selector, entry form, and previous entries.

3. Entry Creation Page

User types journal, uploads media, and saves the entry.

4. Insights Page

Shows user’s mood trends using charts or graphs.
